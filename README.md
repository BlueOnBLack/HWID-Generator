# 🛠️ Windows Licensing & SPP Interop Toolkit

A sophisticated PowerShell suite for low-level interaction with the **Windows Software Protection Platform (SPP)**. This toolkit bypasses high-level managed APIs to interface directly with unmanaged system DLLs, enabling deep inspection, decoding, and cryptographic "forgery" of licensing structures.

## 🚀 Key Features

*   **Cryptographic HWID Binding**: Extracts and parses the raw Computer Hash (Hardware ID) from the system or SPP store. It reconstructs the 0x118-byte HWID structure required to bind a license to a specific machine.
*   **Deep Key Processing**: Implements manual Base24 mapping to bridge the gap between 25-character product keys and their 128-bit (16-byte) binary equivalents, including modern 'N' bit flag handling.
*   **IID Synthesis**: 
    *   Generates **Offline Installation IDs (IID)** by combining the decrypted CD Key metadata (Group ID, Serial, Security ID) with the calculated Computer Hash.
    *   Invokes internal `InnerCall` methods at specific memory offsets, serving as a programmatic alternative to the standard `SLGenerateOfflineInstallationIdEx` API.
*   **Architectural Interop**: Leverages `NativeInteropLib` to manage unmanaged memory pointers and execute functions directly from `pidgenx.dll`, `sppobjs.dll`, and `LicensingWinRT.dll`.
*   **Dynamic Offset**: Direct Call to internal funcation, Find offset dynamically, no fixed offset, support Windows 10, 11 Dll

## 📁 Core Logic Modules

| Module | Responsibility |
| :--- | :--- |
| **Key Encoder** | Base24 string to 16-byte binary conversion. |
| **HWID Parser** | Converts raw WinRT hardware pointers into 64-bit Short-Form Identifiers. |
| **Context Mapper** | Extracts Group ID, Serial, and Security ID from the decrypted key block. |
| **IID Generator** | Orchestrates the binding of Key Info + HWID to produce a valid activation string. |

## 📖 Usage Examples

### Generate Bound Installation ID
```powershell
# 1. Encode the Product Key
$KeyData = Get-PidGenEncoder -ProductKey "XXXXX-XXXXX-XXXXX-XXXXX-XXXXX" -Modern

# 2. Extract Hardware Hash (HWID)
$HWID = Get-ProductHWID -FromStore

# 3. Synthesize the IID (Key Info + Hardware Hash)
$IIDRequest = Invoke-IIDRequest `
    -GroupID $KeyData.GroupID `
    -Serial $KeyData.Serial `
    -SecurityID $KeyData.Security `
    -HWID $HWID.ShortHWID `
    -Mode Retail
```
